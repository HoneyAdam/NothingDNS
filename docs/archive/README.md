# Archive

Bu klasör, geçmiş sürümlerde anlık görüntü olarak yazılmış belgeleri arşivler.
İçerikleri yazıldıkları tarih için doğrudur ancak güncel sürüm durumunu
yansıtmayabilir.

Güncel sürüm bilgisi için [CHANGELOG.md](../CHANGELOG.md)'ye bakın.

Bu dizin şu anda boştur — tüm arşiv belgeleri [docs/legacy/](../legacy/) altına
taşınmıştır.
