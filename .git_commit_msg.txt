api: add cluster join/leave endpoints and cluster manager methods

Add /api/v1/cluster/join (POST) and /api/v1/cluster/leave (DELETE) handlers
in api_cluster.go. Both require operator role and return proper JSON
responses.

Add Cluster.JoinSeed() and Cluster.LeaveCluster() methods to cluster.go.
JoinSeed delegates to gossip.Join in SWIM mode; returns an error for Raft
consensus mode (dynamic joining not supported). LeaveCluster delegates to
StartDraining + CompleteDraining(true).

Route registration added in server.go mux.
